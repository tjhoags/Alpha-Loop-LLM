# Design System (Lightweight)

## Palette
- Primary: #2563eb (blue-600)
- Dark: #0f172a (slate-900)
- Surface: #ffffff
- Border: #e2e8f0 (slate-200)

## Typography
- Font: Inter, system-ui
- Headings: Bold, tight leading
- Body: Regular, 14-16px

## Components
- Buttons: Rounded-lg, solid primary, hover darken
- Cards: Rounded-xl, border slate-200, shadow-sm
- Inputs: Rounded, border slate-200, focus ring blue-500
- Badges: Rounded-full, small, use blue-50/700

## Spacing
- 4px scale; generous padding on forms (12-16px)
