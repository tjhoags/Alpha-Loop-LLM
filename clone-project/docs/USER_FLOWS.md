# User Flows (MVP)

## Organizer
1. Go to dashboard
2. Create meeting type (name, price, duration)
3. Copy booking link `/book/:id`
4. Share link with prospects

## Attendee
1. Open booking link
2. Select slot
3. Enter name/email
4. Confirm (payment stub for now)
5. Receive confirmation message

## Admin (later)
- Manage team routing rules
- Export bookings
- Configure webhooks
